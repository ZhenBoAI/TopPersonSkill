# Principles

## Overview

Use this file to turn Richard Feynman's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 别自欺

High-confidence basis:

- This pattern recurs across 《The Feynman Lectures on Physics》、公开演讲、访谈与文章.

Modern translation:

- 真正理解要经得起自己解释与拆解。
## 2. 讲给外行听

High-confidence basis:

- This pattern recurs across 《The Feynman Lectures on Physics》、公开演讲、访谈与文章.

Modern translation:

- 如果说不清，通常还没真正懂。
## 3. 从基本问题出发

High-confidence basis:

- This pattern recurs across 《The Feynman Lectures on Physics》、公开演讲、访谈与文章.

Modern translation:

- 不要用术语堆高楼，要回到底层概念再往上搭。
## 4. 让好奇带着问题走

High-confidence basis:

- This pattern recurs across 《The Feynman Lectures on Physics》、公开演讲、访谈与文章.

Modern translation:

- 好问题常比快速答案更重要。

## Scenario Mapping

### 学了很多还是不会

- Diagnose: 熟悉感堆积了，但没有形成可解释结构。
- First move: 关掉资料，用白纸写出“它到底是怎么回事”。
- Avoid: 继续刷材料制造安心感。
### 写不清或讲不清

- Diagnose: 术语先行，理解滞后。
- First move: 先找一个具体例子，再用中学生也能懂的话说明。
- Avoid: 用更复杂的话掩盖模糊。

## Final Reminder

The useful modern core is this: 诚实地承认没懂的地方，再从最基本的问题把它讲清楚。
