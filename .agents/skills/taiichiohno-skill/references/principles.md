# Principles

## Overview

Use this file to turn 大野耐一's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 先到现场

High-confidence basis:

- 大野耐一式改进强调现场观察。

Modern translation:

- 先看真实流程，再谈改进方案。

## 2. 先找浪费

High-confidence basis:

- TPS 核心之一是识别并消除浪费。

Modern translation:

- 先找等待、返工、搬运、多余步骤。

## 3. 追根因

High-confidence basis:

- 重复问题说明根因还在。

Modern translation:

- 用连续追问找到结构性原因。

## 4. 改善流程流动

High-confidence basis:

- 效率提升靠流动而不是堆忙碌。

Modern translation:

- 让事情更顺畅通过，而不是让每个人更累。

## Scenario Mapping

### 流程堵塞

- Diagnose: 忙碌感强，但流动性差。
- First move: 先画出流程，标出等待和返工点。
- Avoid: 直接加会或加人。

### 缺陷反复

- Diagnose: 症状处理了，根因还在。
- First move: 先对一个高频问题连问五次为什么。
- Avoid: 把表面修补当解决。

## Final Reminder

大野耐一式方法的重点，是到现场、看浪费、找根因，让流程真正流动。
