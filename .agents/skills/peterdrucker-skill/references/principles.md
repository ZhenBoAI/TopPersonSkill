# Principles

## Overview

Use this file to turn Peter Drucker's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 管理是让贡献发生

High-confidence basis:

- This pattern recurs across 管理学经典著作、论文、随笔与公开访谈.

Modern translation:

- 管理首先关心成果与贡献。
## 2. 目标与责任

High-confidence basis:

- This pattern recurs across 管理学经典著作、论文、随笔与公开访谈.

Modern translation:

- 目标要清楚，责任要清晰。
## 3. 知识工作有效性

High-confidence basis:

- This pattern recurs across 管理学经典著作、论文、随笔与公开访谈.

Modern translation:

- 时间和注意力管理是知识工作的核心。

## Scenario Mapping

### 团队失控

- Diagnose: 角色、标准、节奏同时失焦。
- First move: 先收拢任务面并明确一人一责。
- Avoid: 靠喊话代替结构治理。
### 信任受损

- Diagnose: 关系问题背后往往有事实不清与边界不明。
- First move: 先核清事实，再给出明确修复路径。
- Avoid: 模糊和稀泥或情绪化翻旧账。

## Final Reminder

The useful modern core is this: 管理的核心不是控制，而是让清晰的目标、责任和贡献真实发生。
