# 曾国藩 Skill 说明

## 这个 skill 是做什么的

这个 skill 不是让 AI “扮演曾国藩”，而是把曾国藩在家书、日记、奏折、书信与可信研究里反复出现的做事方法，整理成一套现代可用的判断框架。

它主要适合处理这些问题：

- 自我整顿、拖延、心浮气躁
- 团队混乱、职责不清、执行松散
- 识人、用人、分责、带队
- 合作失信、关系修复、冲突处理
- 长期项目节奏、稳扎稳打、避免冒进
- 家庭秩序、金钱纪律、日常习惯

## 它参考了什么

高置信来源：

- 《曾国藩家书》
- 《曾国藩日记》
- 奏折、批牍、书信、读书札记、诗文
- 《曾国藩全集》

辅助解释来源：

- 唐浩明相关整理与研究
- 张宏杰相关传记与研究
- 年谱、导读、学术研究论文

低置信辅助材料：

- 《冰鉴》
- 《挺经》
- 网络流传名言和商业化二次加工内容

仓库对这些材料的态度是：

`一手资料优先，二手研究辅助，流行金句降权。`

## 怎么用

直接调用：

```text
Use $zeng-guofan to analyze this situation in 曾国藩式做事风格，并给出可执行建议。
```

自然语言触发示例：

- 按曾国藩的风格看，这个团队烂摊子该怎么收拾？
- 如果是曾国藩，会怎么处理合作伙伴失信？
- 用曾国藩的方法帮我定一个三十天自我整顿计划。
- 我该不该现在辞职创业？请用曾国藩式做事法来判断。

## 它通常怎么回答

- 曾式总判
- 先办什么
- 暂不做什么
- 今日 / 七日 / 三十日
- 对人怎么说
- 自省一问

## 它不会做什么

- 不会鼓吹权谋、操控、羞辱、压迫
- 不会把相术、伪托著作、流行金句当成核心依据
- 不会直接照搬晚清历史处境
