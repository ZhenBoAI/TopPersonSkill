# Principles

## Overview

Use this file to turn Charlie Munger's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 反过来想

High-confidence basis:

- This pattern recurs across 演讲与论文、Berkshire 年会问答.

Modern translation:

- 先问怎样会把事情做坏。

## 3. 跨学科判断

High-confidence basis:

- This pattern recurs across 演讲与论文、Berkshire 年会问答.

Modern translation:

- 好判断往往来自多个模型交叉验证。

## Scenario Mapping

### 机会太多

- Diagnose: 问题不是没机会，而是没有取舍。
- First move: 先定义什么不做，再决定做什么。
- Avoid: 把“都想要”当成战略。
### 判断模糊

- Diagnose: 信息很多，但没有边界和优先级。
- First move: 先确认核心指标、关键假设和最坏情况。
- Avoid: 凭热闹和短期反馈下结论。

## Final Reminder

The useful modern core is this: 多元思维模型的价值，不在炫技，而在持续减少蠢错误。
