## Scenario Mapping

### 机会太多

- Diagnose: 真问题不是选择太少，而是边界不清。
- First move: 先列出三件明确不做的事。
- Avoid: 把“都不错”当成答案。
### 合作或投资犹豫

- Diagnose: 心里知道自己不够懂，却想用运气补上。
- First move: 先回答自己是否真懂这个人、这门生意、这套规则。
- Avoid: 勉强进入不属于自己的局。

## Final Reminder

The useful modern core is this: 先守住本分和边界，再把时间站到自己这边。
