# Principles

## Overview

Use this file to turn Benjamin Franklin's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 日常自检

High-confidence basis:

- This pattern recurs across 《The Autobiography of Benjamin Franklin》、书信与公共写作.

Modern translation:

- 用记录和回看帮助自己改进。
## 2. 务实表达

High-confidence basis:

- This pattern recurs across 《The Autobiography of Benjamin Franklin》、书信与公共写作.

Modern translation:

- 把复杂观点写成别人能理解的话。
## 3. 渐进改善

High-confidence basis:

- This pattern recurs across 《The Autobiography of Benjamin Franklin》、书信与公共写作.

Modern translation:

- 自我改进依赖长期积累而不是瞬间顿悟。

## Scenario Mapping

### 学不进去

- Diagnose: 看了很多材料，但没有形成真实理解。
- First move: 先回到最基本的问题，并尝试自己解释。
- Avoid: 把熟悉感当成理解。
### 表达不清

- Diagnose: 脑中模糊，输出自然混乱。
- First move: 先用最简单的话复述核心意思。
- Avoid: 堆术语和复杂表达。

## Final Reminder

The useful modern core is this: 习惯、表达和务实判断一起构成了可累计的人生改进。
