## Scenario Mapping

### 局势不明

- Diagnose: 信息混乱时最怕同时追多个方向。
- First move: 先判断主战场和主矛盾。
- Avoid: 同时展开多个高风险动作。
### 人才取舍

- Diagnose: 真正的问题常不是没人，而是不会取舍和分配。
- First move: 先看忠诚度、稳定性和承担力。
- Avoid: 只看聪明和表现力。

## Final Reminder

The useful modern core is this: 第一性原理有价值，但只有和边界感一起出现时才真正可用。
