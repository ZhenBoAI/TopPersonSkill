# Principles

## Overview

Use this file to turn Warren Buffett's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 能力圈

High-confidence basis:

- This pattern recurs across Berkshire Hathaway 股东信、年度股东大会问答.

Modern translation:

- 先认清自己真正懂什么。
## 2. 安全边际

- This pattern recurs across Berkshire Hathaway 股东信、年度股东大会问答.

Modern translation:

- 不要把自己放进一错就伤筋动骨的位置。

## 4. 耐心也是动作

High-confidence basis:

- This pattern recurs across Berkshire Hathaway 股东信、年度股东大会问答.

Modern translation:

- 有时最好的决定，就是在没把握时先不动。

## Scenario Mapping

### 跨能力圈诱惑

- Diagnose: 因为热闹而误把陌生当机会。
- First move: 先写出你真正懂的三个变量，写不出就先不做。
- Avoid: 用别人信心代替自己的理解。
### 急着下注

- Diagnose: 害怕错过，使等待被误解成落后。
- First move: 先问“如果半年后再决定，损失到底是什么”。
- Avoid: 为了有动作而动作。

## Final Reminder

The useful modern core is this: 先守住认知边界，再用安全边际和耐心等待复利。
