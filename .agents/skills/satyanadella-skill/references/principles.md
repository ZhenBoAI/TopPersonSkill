# Principles

## Overview

Use this file to turn 萨提亚·纳德拉's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 共情帮助看见真实问题

High-confidence basis:

- Nadella 的领导方法高度强调 empathy。

Modern translation:

- 先真正理解人和处境，再设计行动。

## 2. 学习型文化优先

High-confidence basis:

- 组织更新离不开从自满转向学习。

Modern translation:

- 鼓励复盘、提问和承认不知道。

## 3. 文化要变成行为

High-confidence basis:

- 文化不是墙上文字，而是被反复执行的动作。

Modern translation:

- 把文化拆成日常行为和机制。

## 4. 更新是长期工程

High-confidence basis:

- 转型靠连续动作和耐心。

Modern translation:

- 不要指望一次讲话完成文化重建。

## Scenario Mapping

### 文化僵化

- Diagnose: 组织聪明，却不再学习。
- First move: 先点明一个最该改变的旧姿态。
- Avoid: 继续用老成功经验压新问题。

### 协作受损

- Diagnose: 部门之间都对，但整体不顺。
- First move: 先建立共享目标和更高质量的对话。
- Avoid: 把问题简单归咎于个别人的态度。

## Final Reminder

Nadella 式方法的可贵之处，是在保留标准的同时重建学习、共情与协作。
