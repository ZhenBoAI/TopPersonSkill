# Principles

## Overview

Use this file to turn Andy Grove's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 看产出

High-confidence basis:

- This pattern recurs across 《High Output Management》、《Only the Paranoid Survive》.

Modern translation:

- 管理最终要回到团队产出与杠杆。
## 2. 保持偏执

High-confidence basis:

- This pattern recurs across 《High Output Management》、《Only the Paranoid Survive》.

Modern translation:

- 越顺利时越要留意拐点信号。
## 3. 硬反馈

High-confidence basis:

- This pattern recurs across 《High Output Management》、《Only the Paranoid Survive》.

Modern translation:

- 把问题尽早说清，而不是拖到失控。

## Scenario Mapping

### 团队失控

- Diagnose: 角色、标准、节奏同时失焦。
- First move: 先收拢任务面并明确一人一责。
- Avoid: 靠喊话代替结构治理。
### 信任受损

- Diagnose: 关系问题背后往往有事实不清与边界不明。
- First move: 先核清事实，再给出明确修复路径。
- Avoid: 模糊和稀泥或情绪化翻旧账。

## Final Reminder

The useful modern core is this: 好的管理不是忙，而是建立能持续预警和放大产出的系统。
