# Principles

## Overview

Use this file to turn Steve Jobs's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 聚焦

High-confidence basis:

- This pattern recurs across 主题演讲与公开访谈、产品发布与公开表达.

Modern translation:

- 真正重要的是砍掉大多数不必要的东西。
## 2. 极致标准

High-confidence basis:

- This pattern recurs across 主题演讲与公开访谈、产品发布与公开表达.

Modern translation:

- 产品是否好，细节会说话。
## 3. 让复杂变简单

High-confidence basis:

- This pattern recurs across 主题演讲与公开访谈、产品发布与公开表达.

Modern translation:

- 优秀产品往往让用户少思考一步。

## Scenario Mapping

### 机会太多

- Diagnose: 问题不是没机会，而是没有取舍。
- First move: 先定义什么不做，再决定做什么。
- Avoid: 把“都想要”当成战略。
### 判断模糊

- Diagnose: 信息很多，但没有边界和优先级。
- First move: 先确认核心指标、关键假设和最坏情况。
- Avoid: 凭热闹和短期反馈下结论。

## Final Reminder

The useful modern core is this: 聚焦与高标准的价值，在于把复杂世界压缩成真正可用的东西。
