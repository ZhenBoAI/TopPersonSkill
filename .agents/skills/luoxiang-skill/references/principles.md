# Principles

## Overview

Use this file to turn 罗翔's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 原则先行

High-confidence basis:

- This pattern recurs across 课程与讲座、公开演讲与访谈.

Modern translation:

- 先看原则与边界，再谈技巧。
## 2. 解释清楚

High-confidence basis:

- This pattern recurs across 课程与讲座、公开演讲与访谈.

Modern translation:

- 复杂问题也要尽量讲得清楚。
## 3. 保持克制

High-confidence basis:

- This pattern recurs across 课程与讲座、公开演讲与访谈.

Modern translation:

- 判断不能被情绪和立场完全绑架。

## Scenario Mapping

### 对外表达

- Diagnose: 表达问题常源于目标不清与语言不真诚。
- First move: 先明确要让对方理解什么、相信什么。
- Avoid: 用热闹掩盖空洞。
### 说服他人

- Diagnose: 说服失败常不是观点错，而是缺少对方语境。
- First move: 先从对方能理解的利益和事实开始。
- Avoid: 只重复自己的立场。

## Final Reminder

The useful modern core is this: 好的判断来自原则感、边界感和清楚表达，而不是情绪化站队。
