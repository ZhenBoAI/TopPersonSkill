## 1. 算清账

High-confidence basis:

- This pattern recurs across 自传与公开口述材料、采访、演讲与企业公开表达.

Modern translation:

- 先把成本、利润和资源消耗说清楚。

## Scenario Mapping

### 机会太多

- Diagnose: 问题不是没机会，而是没有取舍。
- First move: 先定义什么不做，再决定做什么。
- Avoid: 把“都想要”当成战略。
### 判断模糊

- Diagnose: 信息很多，但没有边界和优先级。
- First move: 先确认核心指标、关键假设和最坏情况。
- Avoid: 凭热闹和短期反馈下结论。

## Final Reminder

The useful modern core is this: 务实不是保守，而是始终用真实成本和真实结果校正自己。
