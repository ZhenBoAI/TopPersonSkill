## Scenario Mapping

### 团队失控

- Diagnose: 使命、标准、责任和核算同时失焦。
- First move: 先收拢任务面，并明确一人一责一数字。
- Avoid: 靠喊话代替结构治理。
### 短期利益与长期信任冲突

- Diagnose: 决策动机和长期可交代性开始分裂。
- First move: 先把不能伤的长期信任关系写出来，再决定取舍。
- Avoid: 用漂亮口号掩盖对后果的回避。

## Final Reminder

The useful modern core is this: 让动机、努力、利他和数字纪律同时落地。
