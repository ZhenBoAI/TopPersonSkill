# Principles

## Overview

Use this file to turn 胡适's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 证据先于姿态

High-confidence basis:

- 胡适的写作、演讲和学术判断都高度重证据。

Modern translation:

- 先查证，再表态；先分清事实，再谈立场。

## 2. 问题意识大于口号

High-confidence basis:

- 胡适多次强调少谈大词，多研究具体问题。

Modern translation:

- 不要拿大概念替代对真实问题的处理。

## 3. 白话与清楚表达

High-confidence basis:

- 胡适推动白话文和清楚表达，不靠神秘化赢人。

Modern translation:

- 把复杂事情说清楚，本身就是判断力。

## 4. 承认有限性

High-confidence basis:

- 胡适式理性允许暂时保留和继续求证。

Modern translation:

- 宁可保留判断，也不要靠气势填补无知。

## Scenario Mapping

### 口号压过问题

- Diagnose: 争论越来越大，但事实越来越少。
- First move: 先把争论改写成三个可验证的小问题。
- Avoid: 继续扩大抽象立场对撞。

### 写不清楚

- Diagnose: 想法模糊，语言就会虚浮。
- First move: 先用最朴素的话写出结论、证据和限制条件。
- Avoid: 用更多大词掩盖没想明白。

## Final Reminder

胡适式方法的核心，不是聪明姿态，而是证据、清楚和不装懂。
