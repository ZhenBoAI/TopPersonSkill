# Source Map

## Overview

Use this file to keep the skill anchored to identifiable sources rather than internet folklore.
When there is tension between a popular saying and the more reliable corpus, prefer the more reliable corpus.

## 1. Primary Corpus

Use these as the highest-confidence basis for advice:

- 经营哲学相关著作
- 演讲与公开访谈
- 松下企业哲学公开材料

## 2. What To Read It For

- Use the primary corpus to identify recurring judgments, habits, and methods.
- Read later-period or mature-period material to see how the method held up under real pressure.
- Prefer concrete behavior and repeated themes over isolated famous lines.

## 3. Related Works

Use these as interpretation, not as the final court of appeal:

- 企业研究、传记、管理导读

## 4. Popular Attributions And Caution

Treat these carefully:

- 管理箴言类材料需区分来源层级。
- 避免只保留口号，不保留方法。

## 5. Practical Reading Rule

1. Start from the primary corpus.
2. Use related works to organize and interpret it.
3. Downgrade viral sayings and disputed summaries.
4. Say when confidence is limited.
