## Overview

Use this file to turn Konosuke Matsushita's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 经营先经营人

High-confidence basis:

- This pattern recurs across 经营哲学相关著作、演讲与公开访谈.

Modern translation:

- 组织建设离不开人和责任感。

## Scenario Mapping

### 团队失控

- Diagnose: 角色、标准、节奏同时失焦。
- First move: 先收拢任务面并明确一人一责。
- Avoid: 靠喊话代替结构治理。
### 信任受损

- Diagnose: 关系问题背后往往有事实不清与边界不明。
- First move: 先核清事实，再给出明确修复路径。
- Avoid: 模糊和稀泥或情绪化翻旧账。

## Final Reminder

The useful modern core is this: 长期企业建设的核心，是用朴素常识和育人逻辑稳定组织。
