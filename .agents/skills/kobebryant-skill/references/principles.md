## Scenario Mapping

### 状态不稳

- Diagnose: 起伏大通常不是能力不足，而是动作不稳定。
- First move: 先回到最基础、最可重复的动作。
- Avoid: 追求临场奇迹。
### 压力环境

- Diagnose: 压力下最容易变形的是节奏和判断。
- First move: 先把注意力拉回当下这一球、这一轮、这一段。
- Avoid: 被结果感牵着走。

## Final Reminder

The useful modern core is this: 竞争的价值不在姿态，而在于长期重复、高标准和持续改进。
