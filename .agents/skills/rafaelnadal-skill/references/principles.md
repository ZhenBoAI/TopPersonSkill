High-confidence basis:

- This pattern recurs across 公开访谈与赛后表达、自传与相关公开材料.

Modern translation:

- 把注意力拉回当下最小任务。

## Scenario Mapping

### 状态不稳

- Diagnose: 起伏大通常不是能力不足，而是动作不稳定。
- First move: 先回到最基础、最可重复的动作。
- Avoid: 追求临场奇迹。
### 压力环境

- Diagnose: 压力下最容易变形的是节奏和判断。
- First move: 先把注意力拉回当下这一球、这一轮、这一段。
- Avoid: 被结果感牵着走。

## Final Reminder

The useful modern core is this: 真正的强，不只是爆发，而是长期稳定、低失误和快速复原。
