## 1. 先生存

High-confidence basis:

- This pattern recurs across 公开讲话与访谈、企业管理相关公开文章与信件.

Modern translation:

- 危机里先保住系统与基本盘。

High-confidence basis:

- This pattern recurs across 公开讲话与访谈、企业管理相关公开文章与信件.

## Scenario Mapping

### 团队失控

- Diagnose: 角色、标准、节奏同时失焦。
- First move: 先收拢任务面并明确一人一责。
- Avoid: 靠喊话代替结构治理。
### 信任受损

- Diagnose: 关系问题背后往往有事实不清与边界不明。
- First move: 先核清事实，再给出明确修复路径。
- Avoid: 模糊和稀泥或情绪化翻旧账。

## Final Reminder

The useful modern core is this: 先保住系统，再用组织和灰度判断穿过危机。
