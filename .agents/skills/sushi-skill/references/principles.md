## Scenario Mapping

### 遭遇挫折

- Diagnose: 受挫后容易把暂时困难误判为整体失败。
- First move: 先稳住节律，再处理情绪与现实。
- Avoid: 在最低谷时做过大决策。
### 情绪起伏

- Diagnose: 情绪波动会放大外部事件。
- First move: 先把情绪与事实分开处理。
- Avoid: 把当下情绪当成最终判断。

## Final Reminder

The useful modern core is this: 在逆境里保住节律、心量和做事能力，而不是只求情绪安慰。
