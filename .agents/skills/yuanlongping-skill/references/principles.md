# Principles

## Overview

Use this file to turn 袁隆平's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 到田里去

High-confidence basis:

- 袁隆平的方法高度依赖现场观察与田野验证。

Modern translation:

- 重要判断先到真实场景验证。

## 2. 长期试验

High-confidence basis:

- 重大成果背后是长期、反复、耐心的实验。

Modern translation:

- 把突破拆成连续试验，而不是一次赌成败。

## 3. 解决真实问题

High-confidence basis:

- 其工作始终指向真实粮食问题和现实需求。

Modern translation:

- 先确认你做的事是否在解决真的痛点。

## 4. 结果重于表态

High-confidence basis:

- 少讲大词，多看结果。

Modern translation:

- 用数据、样本和实际改进说话。

## Scenario Mapping

### 脱离现场

- Diagnose: 方案依赖想象，不依赖真实环境。
- First move: 先去看现场样本，做一个真实验证。
- Avoid: 继续在会议室里空转。

### 长期项目失速

- Diagnose: 没有阶段试验，就容易陷入空耗。
- First move: 先定一个能在本周看到信号的小实验。
- Avoid: 把长期当作模糊拖延。

## Final Reminder

袁隆平式方法最可贵的地方，不是热血，而是耐心、现场感和持续试验。
