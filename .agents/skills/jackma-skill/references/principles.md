# Principles

## Overview

Use this file to turn Jack Ma's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 把话讲懂

High-confidence basis:

- This pattern recurs across 公开演讲与访谈、公开信与创业表达.

Modern translation:

- 复杂的方向要先让人听得懂。
## 2. 市场教育

- This pattern recurs across 公开演讲与访谈、公开信与创业表达.

## 3. 点燃组织

High-confidence basis:

- This pattern recurs across 公开演讲与访谈、公开信与创业表达.

Modern translation:

- 领导者要把方向讲成可行动的共识。

## Scenario Mapping

### 对外表达

- Diagnose: 表达问题常源于目标不清与语言不真诚。
- First move: 先明确要让对方理解什么、相信什么。
- Avoid: 用热闹掩盖空洞。
### 说服他人

- Diagnose: 说服失败常不是观点错，而是缺少对方语境。
- First move: 先从对方能理解的利益和事实开始。
- Avoid: 只重复自己的立场。

## Final Reminder

The useful modern core is this: 表达的价值不在热闹，而在于让市场和组织真正理解并开始行动。
