# Principles

## Overview

Use this file to turn 王阳明's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 知行合一

High-confidence basis:

- This pattern recurs across 《传习录》、书信、奏疏与文集.

Modern translation:

- 判断是否真的理解，要看是否已经落实为行动。
## 2. 事上磨练

High-confidence basis:

- This pattern recurs across 《传习录》、书信、奏疏与文集.

Modern translation:

- 不要躲在概念里，要在具体事情中修正自己。
## 3. 破心中贼

High-confidence basis:

- This pattern recurs across 《传习录》、书信、奏疏与文集.

Modern translation:

- 先处理恐惧、自欺、拖延、好名这些内在阻力。

## Scenario Mapping

### 知道很多却不行动

- Diagnose: 真问题往往不是没道理，而是怕做错、怕难看、怕承担。
- First move: 先选一个今天必须留下痕迹的动作。
- Avoid: 继续收集材料来掩盖迟疑。
### 反复内耗

- Diagnose: 心里同时想赢、想稳、还想不付代价。
- First move: 写下最舍不得放下的执念，再砍掉一个分心源。
- Avoid: 用“再想想”延长拖延。

## Final Reminder

The useful modern core is this: 先把心里的借口和分心拿掉，再把知道的事做出来。
