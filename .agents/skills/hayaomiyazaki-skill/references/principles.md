# Principles

## Overview

Use this file to turn Hayao Miyazaki's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 标准先于速度

High-confidence basis:

- This pattern recurs across 公开访谈与对谈、创作相关随笔与言论.

Modern translation:

- 创作时先保标准，再谈规模。
## 2. 反复打磨

High-confidence basis:

- This pattern recurs across 公开访谈与对谈、创作相关随笔与言论.

Modern translation:

- 好作品往往来自大量返工。
## 3. 保住人味

High-confidence basis:

- This pattern recurs across 公开访谈与对谈、创作相关随笔与言论.

Modern translation:

- 技术与效率不能吃掉作品中的真实感。

## Scenario Mapping

### 创作受阻

- Diagnose: 不是没有想法，而是节律和标准被打乱。
- First move: 先恢复固定工作节律，再谈灵感。
- Avoid: 等待完美状态才开始。
### 质量下滑

- Diagnose: 速度上来了，但作品标准开始松动。
- First move: 先回到最核心的判断标准。
- Avoid: 用产量掩盖质量退化。

## Final Reminder

The useful modern core is this: 创作的核心不是姿态，而是长期标准、返工耐心和真实感。
