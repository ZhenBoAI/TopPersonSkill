## Scenario Mapping

### 创作受阻

- Diagnose: 不是没有想法，而是节律和标准被打乱。
- First move: 先恢复固定工作节律，再谈灵感。
- Avoid: 等待完美状态才开始。
### 质量下滑

- Diagnose: 速度上来了，但作品标准开始松动。
- First move: 先回到最核心的判断标准。
- Avoid: 用产量掩盖质量退化。

## Final Reminder

The useful modern core is this: 节律、耐力和安静积累，比灵感更能决定长期创作。
