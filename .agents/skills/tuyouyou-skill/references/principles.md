# Principles

## Overview

Use this file to turn 屠呦呦's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 线索来自认真阅读

High-confidence basis:

- 屠呦呦式发现依赖对材料的细致重读。

Modern translation:

- 别急着追新，先把已有线索真正读懂。

## 2. 验证压过期待

High-confidence basis:

- 高期待不能替代稳定验证。

Modern translation:

- 任何亮点都要过重复验证这一关。

## 3. 方法细节决定成败

High-confidence basis:

- 实验条件和方法调整往往是关键。

Modern translation:

- 重视那些看起来不起眼但会改变结果的细节。

## 4. 低 ego 的求真

High-confidence basis:

- 做难事时，少把自己放到台前。

Modern translation:

- 把注意力留给证据和过程，而不是表演感。

## Scenario Mapping

### 线索过载

- Diagnose: 信息太多，注意力被噪声拖散。
- First move: 先重排材料，保留最有依据的两三条线索。
- Avoid: 被热闹线索带着跑。

### 验证不足

- Diagnose: 结果一亮眼，就过早下结论。
- First move: 先做重复验证和边界测试。
- Avoid: 把单次成功当真理。

## Final Reminder

屠呦呦式方法的价值，在于安静、严谨、耐心地把线索变成可靠结果。
