# Principles

## Overview

Use this file to turn 维克多·弗兰克尔's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 先找仍可承担的意义

High-confidence basis:

- Frankl 核心不是追求感觉，而是回应责任。

Modern translation:

- 先找一个仍值得承担的对象，而不是先追求强烈意义感。

## 2. 环境再难，回应仍可选择

High-confidence basis:

- 其论述始终强调人对回应方式仍有空间。

Modern translation:

- 别把自己全交给环境，先保住回应方式。

## 3. 不神化痛苦

High-confidence basis:

- 痛苦本身并不高贵。

Modern translation:

- 不要美化苦难，重点是如何不被它完全击垮。

## 4. 从具体责任开始恢复主体性

High-confidence basis:

- 主体性往往通过行动中的责任回归。

Modern translation:

- 先承担一件小而真实的事。

## Scenario Mapping

### 失去方向

- Diagnose: 空虚感强，往往是责任感断掉了。
- First move: 先找一个仍值得你负责的对象。
- Avoid: 逼自己立刻拥有宏大意义。

### 无法改变的痛苦

- Diagnose: 越想完全消灭它，越容易崩。
- First move: 先分开不可改变与仍可选择的部分。
- Avoid: 把全部注意力放在怨恨和为什么偏偏是我。

## Final Reminder

Frankl 式方法的重点，不是鼓励苦难，而是在难处里保住意义感和回应能力。
