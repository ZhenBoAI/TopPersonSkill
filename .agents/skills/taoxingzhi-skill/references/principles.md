# Principles

## Overview

Use this file to turn 陶行知's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 生活即教育

High-confidence basis:

- 陶行知的核心论述始终把生活和教育打通。

Modern translation:

- 设计学习时，先看真实情境和真实任务。

## 2. 教学做合一

High-confidence basis:

- 陶行知反复强调教、学、做不能割裂。

Modern translation:

- 能不能学会，要看是否能做出来、教出来。

## 3. 面向普通人

High-confidence basis:

- 陶行知的教育实践高度面向普通人和基层场景。

Modern translation:

- 方法要能为更多普通人的成长服务，而不是只适合精英。

## 4. 实践中再修正

High-confidence basis:

- 教育不是讲完就结束，而是做过再改。

Modern translation:

- 把反馈和迭代放进学习系统里。

## Scenario Mapping

### 学用脱节

- Diagnose: 知识停留在脑中，没有进入场景。
- First move: 先选一个真实任务，当作这周唯一练习场。
- Avoid: 继续囤课程却不做。

### 训练失效

- Diagnose: 培训很多，但没有形成动作和反馈。
- First move: 先把培训改成练习、示范、复盘三步。
- Avoid: 只靠讲解推动改变。

## Final Reminder

陶行知式方法的价值，在于把学习真正变成做事、做人、解决问题的能力。
