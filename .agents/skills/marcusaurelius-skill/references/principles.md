# Principles

## Overview

Use this file to turn Marcus Aurelius's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 先管自己

High-confidence basis:

- This pattern recurs across 《Meditations》、相关古典史料与时代文献.

Modern translation:

- 先管住判断与反应，再管外物。
## 2. 把情绪和事实分开

High-confidence basis:

- This pattern recurs across 《Meditations》、相关古典史料与时代文献.

Modern translation:

- 别让瞬时感受替代判断。
## 3. 尽责而不戏剧化

High-confidence basis:

- This pattern recurs across 《Meditations》、相关古典史料与时代文献.

Modern translation:

- 做该做的事，不把自己演成主角。

## Scenario Mapping

### 个人拖延与内耗

- Diagnose: 目标不清、想得太多、做得太少。
- First move: 先缩小目标并固定一个最小日课。
- Avoid: 靠情绪冲动做短期猛进。
### 长期习惯建设

- Diagnose: 方法很多，但无法稳定重复。
- First move: 先保留一条最能重复的主线习惯。
- Avoid: 设计过度复杂的系统。

## Final Reminder

The useful modern core is this: 真正有用的斯多葛方法，是自我克制、情绪澄清和安静尽责。
