# Principles

## Overview

Use this file to turn 张瑞敏's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 先拆旧惯性

High-confidence basis:

- 张瑞敏式变革强调先处理旧习惯和旧结构。

Modern translation:

- 别急着上新方案，先砍最阻碍变化的旧东西。

## 2. 责任可见

High-confidence basis:

- 组织要对结果负责，不能只对动作负责。

Modern translation:

- 把责任、结果和反馈周期写清楚。

## 3. 用户导向

High-confidence basis:

- 组织变革最终要回到用户价值。

Modern translation:

- 判断任何组织动作时，先问用户会不会因此更受益。

## 4. 持续自我革命

High-confidence basis:

- 企业不能靠过去成功模式一直吃下去。

Modern translation:

- 定期检查哪种成功经验已经变成包袱。

## Scenario Mapping

### 组织钝化

- Diagnose: 动作很多，但没有真正变化。
- First move: 先点名一个最该被砍掉的旧流程。
- Avoid: 把口号当变化。

### 责任漂浮

- Diagnose: 问题总有人知道，但没有人真正承担。
- First move: 先确定唯一责任人和可验收结果。
- Avoid: 继续多人共担却无人负责。

## Final Reminder

张瑞敏式方法不是喊变革，而是让责任、用户价值和自我颠覆真正落地。
